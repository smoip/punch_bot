# punch_bot

let's learn crystal with punching robots

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  punch_bot:
    github: smoip/punch_bot
```

## Usage

```crystal
require "punch_bot"
```

...and then do some stuff?
